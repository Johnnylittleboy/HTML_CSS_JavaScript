- 主要针对元素的**内容**还有**表单的值**操作。
#### 1.普通元素内容html()[相当于原生innerHTML]
- 获取元素的内容语法规范：
    ```javascript
    html()   // 获取元素的内容
    ```
- 设置元素的内容语法规范：
    ```javascript
    html("内容")  // 设置元素的内容
    ```

#### 2.普通元素文本内容text()[相当于原生innerText]
- 获取元素的文本内容语法规范：
    ```javascript
    text()   // 获取元素的文本内容
    ```
- 设置元素的文本内容语法规范：
    ```javascript
    text("文本内容")  // 设置元素的文本内容
    ```

#### 3.表单的值val()[相当于原生value]
- 获取表单元素的内容语法规范：
    ```javascript
    val()   // 获取表单的值
    ```
- 设置表单元素的内容语法规范：
    ```javascript
    val("表单的值")  // 设置元素的值
    ```
